ReAct is javascript library developed by meta.
To build fast and interactive UI.
ReAct is Single page Application.

Why do we use ReAct?
Problem with javascript full page reload complex dom manipulation. Hard to manage large apps.

Updates only required part, 
Uses reusable components


SPA (Single page Application):
eg. Instagram, Facebook
When you click page does not reload only content loads.

ReAct creates a copy of real dom its called virtual DOM.
When data changes ReAct Updates virtual dom. Updates only changes parts in real dom.   -- Result: High Performance
